# Калькулятор

Простой и красивый веб-калькулятор с поддержкой клавиатуры.

## Возможности

- Базовые математические операции (+, -, ×, /)
- Поддержка десятичных чисел
- Удаление последнего символа
- Очистка экрана
- Поддержка клавиатуры
- Адаптивный дизайн

## Деплой на Vercel

Проект готов к деплою на Vercel. Просто подключите репозиторий к Vercel или используйте CLI:

```bash
vercel
```

## Локальный запуск

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python
python -m http.server 8000

# Node.js (http-server)
npx http-server
```
